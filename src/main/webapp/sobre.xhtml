- Sobre o Projeto

Facelets + Bootstrap - DEMONSTRAÇÃO

Esse projeto foi criado para demonstrar o desenvolvimento de templates utilizando as tecnologias Facelets e Bootstrap.

Nessa demonstração utilizo o JavaServer Faces (JSF) 1.2.


Facelets

Facelets é uma tecnologia desenvolvida para criação e renderização de interface gráfica em projetos que utilizam JavaServer Faces (JSF). Criada como uma alternativa mais aderente ao modelo de componentes imposto pelo JSF.

Bootstrap

É um toolkit para criação de front-end tirando proveito de técnicas recentes para elaboração de design gráfico. Desenvolvido pelo Twitter, o Bootstrap provê tipografia estilizada para diversos componentes HTML.

Tecnologias Open-source

Todas as tecnologias utilizadas para o desenvolvimento desse projeto são Open-source.

O projeto é construído com o Maven. O pom.xml pode utilizado como exemplo para outros projetos que adotam essas e outras tecnologias.
